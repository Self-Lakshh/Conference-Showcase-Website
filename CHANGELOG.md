# Changelog

All notable changes to the ResearchSphere project will be documented in this file.

## [1.0.0] - 2024-05-28
### Added
- Completed master setup.
- Dual-Mode Firebase & LocalStorage Adapter with procedural seed engine.
- Interactive SVG-based citation force-directed Knowledge Graph.
- Personal dashboard displaying conference passes, bookmark logs, and SVG QR tickets.
- Admin CRUD control center for conferences, papers, speakers, and updates.
- Custom canvas-particle Network Loader.
- Outfit and JetBrains Mono modern typography layouts.
- Full viewport responsiveness and accessibility tags.

## [0.5.0] - 2024-04-12
### Added
- Scaffolding of core router and components.
- Tailored color theme setups: aurora violet, neon cyan, emerald green, deep navy.
- Navigation header search synchronizations.
- Speaker profiles, timeline logs, and talk logs.
- Interactive registrations checkouts.
